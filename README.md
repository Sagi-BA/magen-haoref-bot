# 🛡️ מגן על הזכויות — צ'אטבוט זכויות חיילי מילואים

צ'אטבוט מבוסס RAG (Retrieval-Augmented Generation) שמסייע לחיילי מילואים ולמשפחותיהם לקבל מידע על זכויות, מענקים, פיצויים והטבות — על בסיס מסמך הזכויות הרשמי של מרכז החוסן הקהילתי מחוז חיפה (מרץ 2026).

## ⚠️ הבהרה חשובה

הבוט מספק **מידע כללי בלבד** ואינו מהווה ייעוץ משפטי או פיננסי. לאימות זכאות ספציפית יש לפנות למוקד המילואים 1111 שלוחה 4.

## התקנה והרצה מקומית

### 1. שכפול הפרויקט

```bash
git clone <repo-url>
cd magen-haoref-bot
```

### 2. התקנת תלויות

```bash
pip install -r requirements.txt
```

### 3. הגדרת מפתח API

צור קובץ `.streamlit/secrets.toml`:

```toml
OPENAI_API_KEY = "sk-your-key-here"
```

### 4. הרצה

```bash
streamlit run app.py
```

האפליקציה תיפתח בדפדפן בכתובת `http://localhost:8501`.

בהרצה הראשונה, הבוט יבנה את אינדקס ה-FAISS מקובץ `benefits.md` (לוקח כ-10-20 שניות). בהרצות הבאות האינדקס נטען מהדיסק.

## פריסה ל-Streamlit Community Cloud

1. העלה את הפרויקט ל-GitHub (ודא ש-`secrets.toml` נמצא ב-`.gitignore`).

## איך הבוט עובד

### תהליך RAG

1. **Chunking** — קובץ `benefits.md` מפוצל לפי כותרות Markdown (נושא ראשי / משני / תת-נושא). כל chunk מקבל prefix של ההיררכיה שלו לשיפור הרלוונטיות בחיפוש.

2. **Embedding** — כל chunk מומר לוקטור עם `text-embedding-3-small` ונשמר ב-FAISS.

3. **Retrieval** — כשהמשתמש שואל שאלה, נשלפים 5 ה-chunks הכי רלוונטיים.

4. **Generation** — ה-chunks מוזרקים ל-System Prompt יחד עם היסטוריית השיחה, ו-`gpt-4o-mini` מייצר תשובה.

### חוקי ברזל

הבוט מונחה על ידי System Prompt קפדני שכולל:

- **חוק המדרגים** — כל סכום מלווה בציון המדרג הרלוונטי
- **חוק הלינקים** — קישורי הגשה מוצגים בצורה בולטת
- **חוק ההורה היחידני** — זיהוי זכויות יתר
- **חוק התוקף** — ציון תאריך אחרון להגשה
- **חוק המצוקה הנפשית** — זיהוי מילות טריגר והקפצת קווי סיוע מיידית
